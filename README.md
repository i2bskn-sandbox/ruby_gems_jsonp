# RubyGems JSONP

The wrapper of RubyGems API for JSONP.

## Installation

Clone from Github and bundle:

    git clone git://github.com/i2bskn/ruby_gems_jsonp.git
    cd ruby_gems_jsonp
    bundle install --path .bundle --without development test

## Deploy

use Passenger with config.ru.
